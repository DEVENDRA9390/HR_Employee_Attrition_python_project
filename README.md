# HR Employee Attrition Analysis

## Project Overview

This project analyzes employee attrition data to identify the key factors influencing workforce turnover and employee retention. Using Python, the dataset was cleaned, explored, and visualized to uncover actionable insights that can help organizations improve retention strategies and workforce management.

## Objectives

* Analyze employee attrition patterns.
* Identify factors affecting employee turnover.
* Understand the impact of salary, overtime, job role, and satisfaction on attrition.
* Generate data-driven insights for HR decision-making.
* Build visualizations to support workforce analysis.

## Tools & Technologies

* Python
* Pandas
* NumPy

## Project Workflow

### 1. Data Cleaning

* Checked for missing values.
* Removed duplicate records.
* Corrected data types.
* Prepared data for analysis.

### 2. Exploratory Data Analysis (EDA)

* Attrition Distribution Analysis
* Department-wise Attrition
* Job Role Analysis
* Salary and Attrition Relationship
* Overtime Impact on Attrition
* Job Satisfaction Analysis
* Work-Life Balance Analysis

### 3. Data Visualization

* Bar Charts
* Count Plots
* Histograms
* Pie Charts
* Correlation Heatmap

## Key Insights

* Employees working overtime showed higher attrition rates.
* Certain job roles experienced greater employee turnover.
* Lower monthly income was associated with increased attrition.
* Job satisfaction and work-life balance significantly influenced retention.
* Years at company impacted employee turnover patterns.
