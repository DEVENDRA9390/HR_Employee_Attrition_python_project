# HR Employee Attrition Analysis using Python

## Project Overview
This project analyzes HR Employee Attrition data using Python. The objective is to identify key factors affecting employee turnover and provide insights that can help organizations improve employee retention.

## Tools & Libraries Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset Features
The dataset includes:
- Age
- Gender
- Department
- Job Role
- Education
- Monthly Income
- Years at Company
- Work-Life Balance
- Attrition Status

## Project Steps

### 1. Data Loading
- Imported the dataset using Pandas.

### 2. Data Cleaning
- Checked missing values.
- Removed duplicates.
- Verified data types.

### 3. Exploratory Data Analysis (EDA)
- Employee Attrition Distribution
- Department-wise Analysis
- Job Role Analysis
- Income vs Attrition
- Age Distribution
- Correlation Analysis

### 4. Data Visualization
- Bar Charts
- Histograms
- Box Plots
- Count Plots
- Heatmaps

## Key Insights
- Identified departments with higher attrition rates.
- Analyzed the relationship between salary and employee retention.
- Evaluated the impact of work-life balance on attrition.
- Studied employee tenure and turnover patterns.

## Business Recommendations
- Improve employee engagement programs.
- Enhance work-life balance initiatives.
- Review compensation strategies.
- Focus on retention in high-attrition areas.
